Bidirectional LSTM Neural Network

The core of our emotion prediction system is a sophisticated bidirectional LSTM (Long Short-Term Memory) neural network. This architecture excels at processing sequential data like text by maintaining memory of both past and future context.

Architecture Components:

Embedding Layer: Converts words into dense 100-dimensional vectors, allowing the model to understand semantic relationships between words.
Bidirectional LSTM: Processes text in both forward and backward directions with 256 hidden units, capturing complete contextual information.
Dropout Regularization: Prevents overfitting with 50% dropout rate, ensuring the model generalizes well to new text.
Classification Layer: Maps the 512-dimensional LSTM output to 6 emotion categories using a fully connected layer.

Key Advantages:

  • Captures long-range dependencies in text sequences
  • Bidirectional processing provides complete context understanding
  • Robust to varying text lengths and styles
  • Achieves high accuracy across all emotion categories

BERT Tokenization

Our system uses BERT (Bidirectional Encoder Representations from Transformers) tokenization for advanced text preprocessing. This approach handles complex language patterns and ensures robust text understanding.

Tokenization Process:

Subword Tokenization: Breaks text into meaningful subword units, handling out-of-vocabulary words and rare terms effectively.
Special Tokens: Adds [CLS] and [SEP] tokens to mark sentence boundaries and classification points.
Numerical Conversion: Converts tokens to numerical IDs from a 30,522 token vocabulary for neural network processing.
Sequence Normalization: Pads or truncates sequences to 128 tokens for consistent batch processing.

Multi-Class Classification

Our system performs sophisticated multi-class classification to distinguish between six fundamental emotions: sadness, joy, love, anger, fear, and surprise. Each prediction includes detailed probability distributions and confidence metrics.

Classification Process:

Softmax Activation: Converts raw neural network outputs into normalized probability distributions across all six emotion classes.
Cross-Entropy Loss: Optimizes model training using cross-entropy loss with Adam optimizer for effective gradient-based learning.
Confidence Scoring: Provides reliability metrics for each prediction, helping users understand prediction certainty.
Balanced Training: Trained on balanced datasets to ensure fair representation across all emotion categories.
